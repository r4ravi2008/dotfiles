# AI Agent Instructions

You are assisting with software development. Follow these guidelines for all projects.

## General Guidelines

- Write clean, maintainable, well-documented code
- Follow project-specific conventions when they exist
- Prefer simple solutions over complex ones
- Use meaningful variable and function names
- Add tests for new functionality when appropriate

## Code Style

- Follow the existing code style in each project
- Use consistent formatting and indentation
- Prefer composition over inheritance
- Keep functions small and focused

## Git Workflow

- Write clear, descriptive commit messages
- Use conventional commits format when appropriate
- Create focused PRs that address one concern
- Include context in PR descriptions

## Communication

- Be concise but thorough in explanations
- Ask clarifying questions when requirements are ambiguous
- Explain trade-offs when making design decisions
- Reference file paths when discussing code

## Error Handling

- Handle errors gracefully
- Provide helpful error messages
- Log errors appropriately for debugging
- Don't swallow exceptions silently
